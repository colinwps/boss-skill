# 老板工作风格分析器

## 目标

从原材料和用户描述中提取老板的工作风格，输出结构化的"汇报指南"和"决策地图"。

---

## 提取维度

### 1. 决策逻辑

分析老板倾向于哪种决策模式：

| 模式 | 特征信号 |
|------|----------|
| **数据驱动型** | 经常要数据支撑、喜欢看漏斗/指标、不接受"感觉上" |
| **直觉判断型** | 快速拍板、凭经验判断、不喜欢过多分析瘫痪 |
| **共识优先型** | 推动各方对齐后才拍板、重视团队 buy-in |
| **风险规避型** | 先问"有什么坏处"、偏好保守方案、需要多个方案对比 |
| **机会导向型** | 先问"最大机会是什么"、愿意冒险、关注竞争格局 |

### 2. 汇报偏好

- **格式**：PPT / 文档 / 口头 / 邮件
- **结构**：结论前置（先说结果）vs 过程叙事（先说背景）
- **长度**：能接受的最长汇报时间
- **节奏**：喜欢被打断提问 vs 听完再问
- **准备要求**：需要提前对齐 vs 接受临时汇报
- **数据密度**：数字越多越好 vs 点到为止

### 3. 拍板规律

提取以下信息：

**什么情况他会直接拍板**（不需要你再分析）：
- 涉及战略方向
- 金额超过某个阈值
- 影响跨团队协作
- 有竞品参照

**什么情况他会放权给你**：
- 执行层面的细节
- 他不熟悉的技术领域
- 时间紧迫需要快速决定

**什么情况他会推回让你再想**：
- 方案不完整
- 没有数据支撑
- 没有风险评估
- 没有 Plan B

### 4. 关注焦点

从原材料中识别老板最在意的 2-3 个核心关注点：

- 商业结果（GMV / DAU / 利润）
- 技术质量（稳定性 / 架构）
- 团队效率（人效 / 协作）
- 对外形象（行业影响力 / PR）
- 客户满意度
- 成本控制

### 5. 沟通节奏

- **响应速度期望**：消息多久必须回
- **会议偏好**：站会 / 长会 / 1:1 频率
- **同步方式**：主动汇报 vs 被动询问
- **异步工具**：飞书/钉钉/邮件/微信偏好

### 6. 已知雷区

从原材料或用户描述中提取：
- 绝对不能提的话题或项目
- 会让他情绪变坏的触发词
- 他反复强调但下面经常犯的错误
- 前任/离职员工相关禁忌
- 竞品相关敏感点

---
